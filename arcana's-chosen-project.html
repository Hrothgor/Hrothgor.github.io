About this project

This project is my first big and finished game using Unreal 5.

This is a Multiplayer game, you have to be on the same connection, so you may have to use a hamachi server to play with a friend.

This is a fighter game inspired by Smash Bros. The big difference is that it's a card game in same time, you build your deck and in game you will use arcane force to play your cards.

The game is really fast paced.

I am proud of the finished game, because it was really challenging to do a finished game in Multiplayer. I had to be sure every thing was working on client and server side, and when you are new to Multiplayer with Unreal it's not easy at the beginning

Now thanks to this project I have done other project in Multiplayer and it's way faster and easier to setup
