About this project

This project is about creating an immersive environment. The goal of this project was to create a stylized landscape.

Everything in this project has been made by myself, from the 3D models to the textures, the animations, the shaders, the particles, the post-process, the documentation, etc. The only things I took from internet are the sound, they come from freesound.org

When doing this project I already had a bit of experience in game design and level design. The hardest part was the modeling since I am a game programmer.

For this project I tried to use PlayMaker to see how it work and it was interesting but I prefer to use C# since I am more comfortable with it. If I had choice I would 100% do it in C#, it's just way easier and faster I feel.

I am extremely proud of the final result and the skills I was able to acquire. I think the final scene is really immersive and I really like the design which is the main goal of this project.
