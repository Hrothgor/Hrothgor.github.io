About this project

This project is heavly on development, I work every day on it. He is clearly not finished.

I will showcase most of the mechanics and system already implemented, but don't forget that this project is still in development and most of the mechanics are still in development and maybe not finished (like the quest system).

This will be a rpg game where you can shapeshift to different form to achieve your goal, I take a lot of inspiration from World of Warcraft. The main reason is that this is my childhood game and there is big database of it so I can use all the data from World of Warcraft and not waste time on making quest description etc.

I started the development in march and it's going pretty well along my study. I like how is it going and all I am learning from it. Trying to make the most possible generic systems that I could reuse in later games I could do.
