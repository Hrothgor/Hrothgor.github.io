Industrial Scene

About this project

This project is about creating an immersive environment. The goal of this project was to create an industrial scene. I went for a steampunk style with some inspirations from Arcane (Netflix serie).

Everything in this project has been made by myself, from the 3D models to the textures, the animations, the post-process, the documentation, etc. The only things I took from internet are the sound, they come from freesound.org

This project teached me a lot, when doing this project I had no experience in 3D modeling or texturing. I never used Maya or Blender before. But I wanted to be able when I develop my own game to be able to create my own assets, my own world without taking every asset from internet.

Since it was my first time using those software, the final result is not perfect (we can see some resolution problem on some texture and a bit too much hand painted). But at the end I am really proud of the final result and the skills I was able to acquire. I never did 3D modeling before on maya and substance painter and I think the final scene is really immersive and look as I was expected which is the main goal of this project.
